Core Features

  • Full Economy System (Vault integration)
  • Dynamic per-world Scoreboards & Tablist
  • 20+ Admin Commands: vanish, ban, mute, maintenance mode
  • QoL commands: spawn, rtp, warp, home, shop
  • Fully configurable messages, permissions & per-world rules

Customization

Edit plugin behavior, messages, commands, ranks, and per-world rules via configuration files. Example: customize economy settings, scoreboards, or chat formatting.

Integrations

Works seamlessly with bStats for metrics, Vault, LuckPerms, PlaceholderAPI, and most economy/permissions plugins.

Chat & Social

Colorful chat formatting, nickname support, faction chat, global/local channels, and integration with external chat plugins.

Advanced Admin Tools

Teleportation, vanish, logs, warnings, maintenance mode, and detailed player info commands.
